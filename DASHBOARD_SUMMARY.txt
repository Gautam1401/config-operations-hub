OVERVIEW
================================================================================

The ARC Configuration Dashboard is a Streamlit-based interactive web application
designed for the Tekion Config Operations team to track and manage ARC 
configuration activities across multiple regions and modules.

Key Features:
✓ Interactive KPI cards (Total Go Live, Completed, WIP, Not Configured)
✓ Region-based filtering (NAM, EMEA, APAC, LATAM)
✓ Module breakdown (Service, Parts, Accounting)
✓ Dynamic date filters (Current Month, Next Month, YTD)
✓ Real-time data filtering and drill-down
✓ CSV export functionality
✓ Responsive design with teal/white color scheme

================================================================================
DATA FLOW
================================================================================

1. User Access
   └─> Opens Config Operations Hub (http://localhost:8502)

2. Dashboard Load
   └─> Clicks "ARC Configuration" tab
       └─> render_arc_dashboard() called

3. Data Loading
   └─> load_data() fetches from SharePoint or Mock
       └─> ARCDataProcessor wraps DataFrame
           └─> Data prepared (dates converted, calculations done)

4. Initial Display
   └─> Sidebar shows filters
   └─> Data tab shows:
       ├─> KPI cards (4 cards)
       ├─> Region banners (4 regions)
       └─> Data table

5. User Interaction
   └─> Click KPI card
       └─> Session state updated
           └─> Data filtered by status
               └─> Module breakdown shown
                   └─> Table updated

6. Further Filtering
   └─> Click Region banner
       └─> Data filtered by region
           └─> Counts updated
               └─> Table refreshed

7. Export
   └─> Click Download button
       └─> CSV file generated with filtered data

================================================================================
SESSION STATE VARIABLES
================================================================================

st.session_state.selected_kpi       → Currently selected KPI/Status
st.session_state.selected_region    → Currently selected region
st.session_state.selected_lob       → Currently selected module
st.session_state.date_filter        → Current date filter selection
st.session_state.data_loaded        → Boolean flag for data load status

================================================================================
CONFIGURATION OPTIONS
================================================================================

Data Source:
  USE_MOCK_DATA = True/False          (Toggle between mock and SharePoint)

Mock Data:
  MOCK_DATA_ROWS = 100                (Number of mock records)

Regions:
  REGIONS = ['NAM', 'EMEA', 'APAC', 'LATAM']

Status Options:
  STATUS_OPTIONS = ['Completed', 'WIP', 'Not Configured']

Modules:
  LOB_OPTIONS = ['Service', 'Parts', 'Accounting']

Colors:
  PRIMARY_COLOR = "#008080"           (Teal)
  SECONDARY_COLOR = "#FFFFFF"         (White)

Date Filters:
  - Current Month (Dynamic: "October 2025")
  - Next Month (Dynamic: "November 2025")
  - YTD (Year to Date)

================================================================================
RUNNING THE DASHBOARD
================================================================================

Local Development:
  $ streamlit run config_operations_hub.py --server.port=8502

Standalone ARC Dashboard:
  $ streamlit run arc_dashboard/app.py --server.port=8501

Access:
  Local:    http://localhost:8502
  Network:  http://192.168.0.108:8502
